# Applied Econometrics Projects

Repository built to complete exercises and projects from the Applied Econometrics course.

---

## Project description

Weekly applied exercises using two datasets:

- **Airbnb listings** (weeks 1вЂ“2): price analysis, OLS regression, AI-generated neighbourhood scores
- **US state-year fatality panel** (week 3+): panel data methods

## Data flow

```

  в†’ week_1/build/code/clean_data_task1.R
      в†’ datasets/airbnb/processed/listings_{region}_cleaned.rds
          в†’ week_2/build/code/data_prep.R
              в†’ datasets/airbnb/processed/listings_{region}_week2.rds
                  в†’ week_2/analysis/code/data_analysis.R  (+ neigh_scores/)
                      в†’ datasets/airbnb/processed/listings_{region}_final.rds

datasets/US_state_year/fatality.dta
  в†’ week_3/build/code/data_setup.R
      в†’ datasets/US_state_year/processed/fatality_panel.rds
```

All processed data lives in `datasets/*/processed/` and is gitignored (reproducible from scripts).

## Tracking policy (source-first)

**Tracked in Git**
- Source files (`.R`, `.qmd`)
- Configuration files (`datasets/*/_config_*.R`)
- Documentation and reports source text
- Canonical raw inputs in `datasets/`

**Ignored in Git (generated artifacts)**
- `week_*/report/*.html`
- `week_*/report/*.pdf`
- `week_*/analysis/output/figures/*.png`
- `week_*/analysis/output/tables/*.md`
- `datasets/*/processed/`

All ignored artifacts are reproducible from scripts and local rendering.

## How to change region / parameters

**Airbnb analysis (weeks 1вЂ“2):** edit `REGION` in `datasets/airbnb/_config_airbnb.R`

```r
REGION <- "Milan"   # any city in datasets/airbnb/listings/
```

**Fatality analysis (week 3+):** edit `YEAR_RANGE` in `datasets/US_state_year/_config_fatality.R`

```r
YEAR_RANGE <- c(1982, 1988)
```

## AI-generated neighbourhood scores

For week 2, neighbourhood-level scores (Coolness, Centrality, Quietness, Fanciness) were
generated manually using three AI agents with identical prompts:

| Agent | Weight | File |
|-------|--------|------|
| ChatGPT | 0.25 | `datasets/airbnb/neigh_scores/puglia_scores_chatgpt.csv` |
| Gemini (Thinking) | 0.25 | `datasets/airbnb/neigh_scores/puglia_scores_gemini.csv` |
| Perplexity Deep Research | 0.50 | `datasets/airbnb/neigh_scores/puglia_scores_perplexity.csv` |

**Source-of-truth**
- Score CSVs: `datasets/airbnb/neigh_scores/`
- Prompt and process documentation: `week_2/analysis/input/agents/`

## Reproducing results

```r
source("run_all.R")
```

Then render reports locally when needed:

```bash
quarto render week_1/report/Week_1.qmd

```

Or run individual scripts in order (see `run_all.R` for the sequence).
